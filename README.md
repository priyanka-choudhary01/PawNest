# PawNest – Full Stack E-commerce Platform

A MERN-based e-commerce platform with authentication, cart, and order management.

## Features
- JWT Authentication with OTP verification
- Cart and checkout system
- Order management APIs

##  Architecture
- MVC backend structure
- REST API design
- Modular backend logic

##  Tech Stack
React, Node.js, Express, MongoDB

##  Live Demo
https://paw-nest-47s9.vercel.app/

## ⚙️ Setup
git clone <repo>
npm install
npm start
